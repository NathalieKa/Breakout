# Breakout

🎮 Breakout

Eine erweiterte Breakout-Variante, entwickelt in Python mit Pygame. Das Projekt entstand im Rahmen des Moduls Skriptsprachen und verbindet klassische Spielmechanik mit modernen Features und einem erweiterten Spielerlebnis.

📖 Spielprinzip

Der Spieler steuert einen Schläger horizontal am unteren Bildschirmrand. Ein Ball prallt von den Wänden und dem Schläger ab und zerstört dabei Blöcke. Ziel ist es, alle Blöcke zu zerstören, um ins nächste Level zu gelangen. Das Spiel endet, wenn der Ball den unteren Bildschirmrand berührt.

✨ Features

Blöcke

Standard-Blöcke – werden durch einen Treffer zerstört
Explosive Blöcke – zerstören beim Treffer alle benachbarten Blöcke
Unzerbrechliche Blöcke – können nicht zerstört werden und erfordern besondere Strategien
Prismenblöcke – ändern die Richtung oder Geschwindigkeit des Balls auf unvorhersehbare Weise

Power-Ups
Blöcke können beim Zerstören Power-Ups freisetzen:

Schläger vergrößern
Ballgeschwindigkeit verändern
Zusätzliche Bälle ins Spiel bringen

Spielmodi

Mehrstufige Level – jedes Level bietet eine einzigartige Anordnung und Kombination von Blöcken
Überlebensmodus – der Ball beschleunigt kontinuierlich; Ziel ist es, so lange wie möglich zu überleben
God-Mode – ermöglicht das Erkunden des Spiels ohne Risiko des Scheiterns

Punktesystem & Bewertung

Punkte für jeden zerstörten Block
Bonuspunkte für das Abschließen eines Levels ohne Ballverlust
Sternbewertungssystem – visuelle Leistungsbewertung pro Level (1–3 Sterne)

Benutzeroberfläche

Anzeige von Punktzahl, aktueller Levelnummer und verbleibenden Leben
Optionales Tutorial für neue Spieler (erklärung von Steuerung, Power-Ups und Spezialblöcken)
Musik und Soundeffekte
